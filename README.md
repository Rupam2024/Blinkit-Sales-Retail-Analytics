# 🛒 Blinkit Sales & Customer Data Analysis

## 📌 Project Overview

This project focuses on analyzing Blinkit's retail sales data to uncover valuable business insights related to product performance, outlet performance, customer purchasing behavior, and sales trends.

Using Python, SQL, and Power BI, the raw dataset was cleaned, transformed, analyzed, and visualized through an interactive dashboard that helps stakeholders make data-driven decisions.

---

## 🎯 Objectives

- Analyze overall sales performance.
- Identify top-performing products and categories.
- Evaluate outlet-wise sales and customer trends.
- Discover factors affecting product sales.
- Create an interactive dashboard for business decision-making.

---

## 📊 Dataset Information

The dataset contains information related to:

- Product Details
- Product Categories
- Sales Revenue
- Outlet Information
- Outlet Size & Location
- Item Visibility
- Item Weight
- Establishment Year
- Customer Purchase Trends

### Key Columns

- Item Identifier
